# Abu Portfolio

A modern personal portfolio website for Jafeer Abdullah, built to showcase web development, mobile app development, UI/UX design, graphic design, services, featured projects, and contact information in a clean responsive layout.

## Live Preview

After deployment, add your live portfolio link here:

```text
https://jafeerabdullah.github.io/abu-portfolio/
```

## About The Project

This portfolio is designed as a professional online presence for presenting skills, services, and selected work. The home page includes an animated AI-style background generated from the profile photo, giving the hero section a premium and personal visual identity.

The website is fully static, so it can be hosted easily on GitHub Pages, Vercel, Netlify, or any static hosting platform.

## Features

- Responsive portfolio design for desktop, tablet, and mobile
- Animated hero section with profile-photo-based AI visual background
- Role rotator text animation
- About section with profile image and experience highlights
- Skills and expertise cards
- Services section
- Featured projects section
- Contact form UI with client-side validation
- Smooth scrolling navigation
- Mobile hamburger menu
- Scroll reveal animations
- Clean dark theme with modern gradients

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome icons
- Canvas animation

## Getting Started

To run this project locally, download or clone the repository and open `index.html` in your browser.

```bash
git clone https://github.com/jafeerabdullah/abu-portfolio.git
cd abu-portfolio
```

Then open:

```text
index.html
```

You can also run a local server:

```bash
python -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```

## Deployment

This portfolio can be deployed with GitHub Pages.

1. Push the project to GitHub.
2. Open the repository on GitHub.
3. Go to **Settings**.
4. Open **Pages**.
5. Select **Deploy from a branch**.
6. Choose the `main` branch and `/root` folder.
7. Save the settings.

Your website will be available at:

```text
https://jafeerabdullah.github.io/abu-portfolio/
```
